\exercise{Min e Max}
Dotare la classe \cod{Lista} (\cfr{Ex:Lista Semplicemente Collegata}) del metodo \cod{MinMax()} secondo la seguente specifica.

\begin{methodslist}

\method{MinMax}{\emptyset}{TElem,TElem} {

}

\end{methodslist}
