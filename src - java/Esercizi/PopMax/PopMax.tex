\exercise{PopMinMax}
Dotare la classe \cod{Coda} (\cfr{Ex:Coda}) dei metodi \cod{popMax()} e \cod{popMin()} secondo la seguente specifica.

\begin{methodslist}

\method{popMax}{int}{int} {
Detto \cod{n} il valore del parametro di ingresso di tipo intero, il metodo estrae i primi \cod{n} valori di testa della struttura e restituisce il massimo tra questi. In caso di coda vuota il comportamento di questo metodo è non specificato.
}

\method{popMin}{int}{int} {
Detto \cod{n} il valore del parametro di ingresso di tipo intero, il metodo estrae i primi \cod{n} valori di testa della struttura e restituisce il minimo tra questi. In caso di coda vuota il comportamento di questo metodo è non specificato.
}

\end{methodslist}
