Dotare la classe \cod{AlberoBinario} (\cfr{Ex:Albero Binario}) del metodo \cod{contaMinMax()}, secondo la seguente specifica.
  
\begin{methodslist}

\method{contaMinMax}{int,int}{int} {

}
