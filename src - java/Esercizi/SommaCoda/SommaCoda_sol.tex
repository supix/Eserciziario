\thesolution{Somma Coda}
L'approccio più efficiente per risolvere questo problema consiste nel tenere memoria del valore della coda in un membro privato della lista. Tale valore deve essere costantemente aggiornato, a cura di tutti i metodi che possono potenzialmente alterarlo: inserimento, eliminazione, svuotamento, ecc. Si noti che lo stesso metodo \cod{sommaCoda()} finisce per alterare il valore della coda. Di seguito si mostrano le implementazioni dei metodi \cod{inserisci()} e \cod{sommaCoda()}, nelle ipotesi che la lista sia dotata di una variabile-membro privata definita come segue:

\begin{codequote}
	public class Lista {
		...
		private int valoreCoda;
		...
	};
\end{codequote}
