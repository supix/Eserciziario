\exercise{Elimina Pari e Dispari}
Dotare la classe \cod{Lista} (\cfr{Ex:Lista Semplicemente Collegata}) dei metodi \cod{eliminaElPostoPari()} ed \cod{eliminaElPostoDispari()}, secondo la seguente interfaccia.

\begin{methodslist}

\method{eliminaElPostoPari}{\emptyset}{int} {
Elimina dalla lista tutti gli elementi di posto pari \cod{(0, 2, 4, ...)}. Restituisce il numero di elementi eliminati.
}

\method{eliminaElPostoDispari}{\emptyset}{int} {
Elimina dalla lista tutti gli elementi di posto dispari \cod{(1, 3, 5, ...)}. Restituisce il numero di elementi eliminati.
}

\end{methodslist}
